Plugin
====================================

Click ``Plugins`` button in right toolbar to open plugins interface.

.. image:: ../../display/plugins.png
    :alt: plugins.png

.. tip:: All ISAT plugins are released in the form of Python packages. After installing the plugin, restart ISAT to load the plugin.

.. danger:: Activating multiple plugins simultaneously may cause conflicts.
